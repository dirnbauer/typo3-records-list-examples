.. _installation:

============
Installation
============

Install the extension with Composer:

.. code-block:: bash
    :caption: Composer installation

    composer require webconsulting/records-list-examples

Activate the extension:

.. code-block:: bash
    :caption: Activate extension

    ./vendor/bin/typo3 extension:activate records_list_examples

.. _installation-requirements:

Requirements
============

This release supports TYPO3 v14 only:

* TYPO3 CMS 14.3 or later
* PHP 8.2 through 8.5
* EXT:records_list_types 14.x

TYPO3 v13 support has been dropped.
